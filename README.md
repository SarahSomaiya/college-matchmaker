# CollegeXed – Smart College Comparison & Discovery Platform
CollegeXed is a modern web application designed to help students explore, compare, and discover colleges through an intuitive and interactive interface. The platform provides a clean UI, smart comparison features, and a responsive experience across devices.
This project was developed as part of the Project-Based Learning (PBL) coursework.


## Live Website
https://SarahSomaiya.github.io/college-matchmaker/


## Project Overview
CollegeXed allows users to:
- Browse and explore colleges
- Search and filter colleges
- Compare colleges side-by-side
- Toggle between dark and light mode
- Experience responsive UI across devices
- Navigate seamlessly using modern routing

The goal of this project is to simplify the college discovery process and provide students with an easy way to evaluate options.


##  Tech Stack
- **Frontend Framework:** React (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** ShadCN UI
- **Routing:** React Router
- **Deployment:** GitHub Pages

## Deployment

The application is deployed using GitHub Pages.

Build the project:npm run build
Deploy the project:npm run deploy
Live URL:https://sarahsomaiya.github.io/college-matchmaker/
Navigate to the project folder:cd college-matchmaker
Install dependencies:npm install
Start the development server:npm run dev
